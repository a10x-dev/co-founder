# Changelog

## v0.3.1 — 2026-03-08

### New Features
- **Telegram integration** — Connect your Telegram account to chat with your co-founder on the go. Send text messages and photos directly from Telegram.
- **Telegram photo support** — Send screenshots and images via Telegram; the co-founder can see and analyze them.
- **Telegram settings panel** — Configure and manage your Telegram bot connection from the Settings tab.

### Fixes
- **Session hover effect** — Hover highlight now only applies to session header rows, not expanded content.

## v0.3.0 — 2026-03-07

### New Features
- **Deliverables tab** — Browse, view, and dismiss deliverables from `.founder/` directory.
- **App version display** — Settings page now shows the current app version.

## v0.2.4 — 2026-03-07

### Fixes
- Auto-update signing key fix — `.sig` files now generated correctly for updater.

## v0.2.3 — 2026-03-06

### New Features
- Role-based session templates (6 roles).
- Screenshot feedback modal with Slack integration.
